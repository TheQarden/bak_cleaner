# 🗑️ bak_cleaner

Simple Python script to delete all `.bak` files recursively. Works on Windows, Linux, macOS, and PyDroid (Android).

## ✨ Features
- Recursively deletes all `.bak` files
- Shows count of files before deletion
- Requires confirmation before deleting
- Cross-platform

## 📥 Installation

```bash
git clone https://github.com/theqarden/bak_cleaner
cd bak_cleaner
python3 bak_cleaner.py
